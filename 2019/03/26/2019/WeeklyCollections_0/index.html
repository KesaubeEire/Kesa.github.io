本期 all from 阮一峰老师的每周总结.

阮一峰老师的每周分享总结帖

这个东西太爆炸了
我基本上就指望这个东西过活了

也许阮一峰老师的一些技术思想和教程有一定的偏差
但是他个人对整个华语编程界的贡献是很大的
他的教程给无数人带来方便
激励了无数人开启博客式的编程生涯
已经是个值得尊重的人了

更遑论阮一峰老师的思想之高和阅读之深广
在泛 IT 领域的著作兼译作都有成就
真的是一个活出了时代精神的程序员榜样

在线 IDE

最近找了好几个在线的 IDE
一方面是 zwj 需要做一个前端的东西
一方面是正好碰到一些这种东西
我觉得很不错

1.emun

  • 优点 : 很好看
  • 缺点 : 动态不够,需要自己拉拽

2.Code Box

  • 优点:一个搞了很多年的 Cloud IDE,功能齐全

网络安全教程

1.Hacker Tools

是一个全英文的 Hacker 入门指导.
有看起来丰富的资源和选项
但是我觉得毕竟是英文理解还是不够舒服

2.Micro8

MarkDown 转微信小程序

这是一个非常可爱的项目
这个需求我觉得是一个程序员很有可能面临的需求
有大佬搞定那当然是极好的

可视化算法

1.Algorithm Visualizer

可视化的算法演示器,既有 Web IDE 的功能,又有可视化算法,非常牛逼

2.LeetCode Animation All in One

作者的宏图大志值得学习~

Shell 相关语句查询解释

1.ExplainShell.com

这个是个英文的复杂语句解释
可以解释长长的复杂 shell 语句
并针对每个 -x –xx 等进行解释
基本上就是一个 man 和 -help 的集合应用

2.Linux 小程序

图形学小入门

Learn Computer Graphics From Scratch!

一个看起来比较入门但是我觉得非常值得我去看的网站
应该是比起我的图形学圣经一样有价值的教程

Java 相关

1. MyPerf4J

这是一个 Java 的可视化监控库

2. 为这一个功能下 IDEA

Jetbrain 的 IDEA 出的一个 java 的 UML
非常之漂亮.
看到这个我惊了

想到了我和文杰当时找到的一个叫 understand 的 C#UML 类图

也是一个很棒的软件啦
但是看到 JetBrain 出的东西一定要搞一下
JetBrain 在我心里的地位真的是很高
教育免费产品随便用 各种工具既好看又效率
真的是一家非常务实而有气质的公司
是我的目标没错了
